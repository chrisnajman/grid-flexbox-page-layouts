Main

The table is min-width:700px.

The table is enclosed by a div with CSS style of overflow:auto. This ensures basic accessibility: when the screen gets too small, you can horizontally scroll the table content.

However, I ran into a bug when using .page-layout-grid: at screen sizes less than 700px, instead of the scrollbars displaying at the bottom of the table container, they appeared on the viewport (causing the whole page to scroll).

The bug was removed by using the CSS rule .main-is-page-layout-grid{overflow:hidden} on main.
